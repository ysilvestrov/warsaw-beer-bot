# Project: Warsaw Beer Crawler Bot
- **Stack**: Node.js, TypeScript, Telegraf (Telegram API), SQLite (база даних).
- **Style**: Функціональне програмування, модульна структура. Усі ключі доступу (API) мають читатися з файлу `.env`.
- **Workflow**: Використовуй автономний цикл (brainstorming -> writing-plans -> worktrees).
- **Testing**: Кожен новий модуль логіки має бути покритий базовими тестами (Jest) перед злиттям гілки.
