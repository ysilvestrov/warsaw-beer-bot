# Flasker brewery/name split — generated registry Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace Flasker's naive first-token brewery split with a generated 49-brewery registry so two-word / Cyrillic / guest breweries resolve correctly (issue #304).

**Architecture:** Keep the existing curated `resolveBreweryRule` (5 rules: Mad Brew, VibrantPour, Copper Head, Flasker, Hoppy Hog) as the first, highest-trust path — untouched, so all existing tests stay green. Layer a generated brewery **registry** (the other 49 site breweries, `{ match[], canonical }`) beneath it: resolve by product **tag**, then by **title-head** prefix, then fall back to today's `splitBreweryName`. A committed generator script regenerates the registry from Flasker's live brand strip.

**Tech Stack:** TypeScript (ESM), Vitest, Playwright (generator only, lazy-imported), cheerio-free DOM parsing via the extension's existing selectors.

**Spec:** `docs/superpowers/specs/2026-07/2026-07-18-flasker-brewery-split-design.md`

---

## Context for the implementer

- All paths below are under `extension/` (its own package: `type: module`, `npm test` = `vitest run`, `npm run typecheck` = `tsc --noEmit`). Run commands from `extension/`.
- The brewery is resolved inside `parseTitle` in `extension/src/sites/flasker.ts`. Read that file and `extension/src/sites/flasker.test.ts` first.
- Existing helpers you will reuse (do not reimplement): `stripTitleAlias(head, aliases)` strips a leading brewery form (and separators) from the title head to yield the name; `stripMerchandisingPrefix(name)` removes `ПРЕДРЕЛІЗ`/`ПРОБНИК:` labels; `splitBreweryName(head)` is the naive fallback; `normalizeTag(tag)` lowercases/trims a tag.
- The registry deliberately **excludes** the 5 breweries already covered by `resolveBreweryRule`. This avoids breaking the "conflicting curated tags → fall back" tests (57, 64), which depend on those breweries being recognized only by the curated rules.
- Resolution order after this change (first hit wins): **(1)** `resolveBreweryRule` (curated) → **(2)** registry∩tags → **(3)** registry∩title-head → **(4)** `splitBreweryName` fallback.

```ts
// Registry path: resolve a brewery from the product's own tags. Returns null when
// no tag is a known registry brewery, or when two *different* breweries tie
// (ambiguous collab) — the caller then falls through to title-head / fallback.
export function breweryFromRegistryTags(tags: string[]): FlaskerBrewery | null {
  const set = new Set(tags.map(normalizeTag));
  const hits = FLASKER_BREWERIES.filter((b) => b.match.some((m) => set.has(m.toLowerCase())));
  return hits.length === 1 ? hits[0] : null;
}

// Registry path: resolve a brewery that appears as the leading prefix of the title
// head. Longest match wins (so "Хмільний кіт" beats a bare "Хмільний"). Requires a
// word boundary (exact head or `<brewery> `) so "DUMArine" never matches "DUMA".
export function breweryFromRegistryHead(
  head: string,
): { brewery: FlaskerBrewery; matched: string } | null {
  const lower = head.toLowerCase();
  let best: { brewery: FlaskerBrewery; matched: string } | null = null;
  for (const brewery of FLASKER_BREWERIES) {
    for (const m of brewery.match) {
      const lm = m.toLowerCase();
      if (lower === lm || lower.startsWith(`${lm} `)) {
        if (!best || m.length > best.matched.length) best = { brewery, matched: m };
      }
    }
  }
  return best;
}
```

Create `extension/scripts/gen-flasker-breweries.ts`. Playwright is **lazy-imported inside `main()`** so importing this module in the test does not load Playwright:

```ts
import { writeFileSync } from 'node:fs';
import { fileURLToPath } from 'node:url';
import type { FlaskerBrewery } from '../src/sites/flasker-breweries.generated';

// Non-brewery site sections + the 5 breweries handled by the curated BREWERY_RULES.
const DROP = new Set([
  'Імпортне пиво', 'НАБОРИ І КОЛАБИ',
  'Mad Brew', 'Vibrant Pour', 'Copper Head', 'Flasker', 'Hoppy Hog',
]);

// Curated canonical overrides (transliteration / abbreviation / spacing / catalog
// form) where the Flasker display name will not reach the real Untappd brewery.
// Seeded from the #304 catalog reconciliation; extend as enrichment misses surface.
const OVERRIDE: Record<string, string> = {
  'Berryland': 'BERRYLAND', 'Gold Fish': 'GoldFish', 'Heather House': 'Heather House Brewery',
  'KLB': 'Kyiv Local Brewery', 'Markel': 'Markel Brewery', 'Mova': 'MOVA brewing co.',
  'MUZA': 'MUZA BREWING CO', 'ProArt': 'ProArt Brewery', 'REALIST': 'Realist',
  'REBREW': 'Rebrew', 'RED CAT': 'Red Cat', 'Ten Men': 'Ten Men Brewery',
  'The Bad Beaver': 'The Bad Beaver!', 'Underwood': 'Underwood Brewery', 'Volta': 'Volta Brewery',
  'Дідько': 'Дідько Brewery', 'Звір Beer': 'Zvir Beer', 'Оболонь': 'Obolon', 'Правда': 'Pravda',
};

// Extra match forms (spacing variants seen in titles/tags).
const EXTRA_MATCH: Record<string, string[]> = {
  'Gold Fish': ['GoldFish'],
  'Vibrant Pour': ['VibrantPour'], // harmless; Vibrant Pour is dropped, kept for documentation
};

function decodeEntities(s: string): string {
  return s
    .replace(/&#0?39;/g, "'").replace(/&apos;/g, "'").replace(/&quot;/g, '"')
    .replace(/&amp;/g, '&').replace(/\s+/g, ' ').trim();
}

export function buildRegistryEntries(rawDisplayNames: string[]): FlaskerBrewery[] {
  const seen = new Set<string>();
  const entries: FlaskerBrewery[] = [];
  for (const raw of rawDisplayNames) {
    const display = decodeEntities(raw);
    if (!display || DROP.has(display)) continue;
    const key = display.toLowerCase();
    if (seen.has(key)) continue;
    seen.add(key);
    entries.push({
      match: [display, ...(EXTRA_MATCH[display] ?? [])],
      canonical: OVERRIDE[display] ?? display,
    });
  }
  return entries.sort((a, b) => a.match[0].localeCompare(b.match[0]));
}

function render(entries: FlaskerBrewery[]): string {
  const rows = entries.map((e) =>
    `  { match: [${e.match.map((m) => JSON.stringify(m)).join(', ')}], canonical: ${JSON.stringify(e.canonical)} },`,
  ).join('\n');
  return `// GENERATED by scripts/gen-flasker-breweries.ts — do not edit by hand.
// Source: Flasker brand strip (mb-brand-tile), reconciled against the Untappd
// catalog for canonical forms. Regenerate with: npm run gen-flasker-breweries
export interface FlaskerBrewery {
  /** Display forms as they appear in Flasker titles/tags (case-insensitive match). */
  match: string[];
  /** Brewery emitted to the matcher (canonical Untappd form where known). */
  canonical: string;
}

export const FLASKER_BREWERIES: FlaskerBrewery[] = [
${rows}
];
`;
}

async function main(): Promise<void> {
  const { chromium } = await import('playwright');
  const browser = await chromium.launch({ headless: true });
  try {
    const page = await browser.newPage({ userAgent: 'Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36' });
    await page.goto('https://flasker.com.ua/', { waitUntil: 'networkidle', timeout: 60_000 });
    await page.waitForSelector('.mb-brand-tile img[alt]', { timeout: 15_000 });
    const names = await page.$$eval('.mb-brand-tile img[alt]', (imgs) =>
      imgs.map((img) => (img as HTMLImageElement).getAttribute('alt') ?? ''));
    const entries = buildRegistryEntries(names);
    const outPath = fileURLToPath(new URL('../src/sites/flasker-breweries.generated.ts', import.meta.url));
    writeFileSync(outPath, render(entries), 'utf8');
    console.log(`Wrote ${entries.length} breweries to src/sites/flasker-breweries.generated.ts`);
  } finally {
    await browser.close();
  }
}

// Run only when invoked directly, not when imported by the test.
if (process.argv[1] && fileURLToPath(import.meta.url) === process.argv[1]) {
  main().catch((e) => { console.error(e); process.exit(1); });
}
```
